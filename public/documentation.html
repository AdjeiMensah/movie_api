API Documentation

Request Method URL Endpoint Description Query Parameters Data Expected in Request Body Response
GET /movies Get a list of all movies. - - Array of movie objects
GET /movies/:title Get data about a single movie by title. title - Movie object
GET /movies/genres/:name Get data about a genre by name/title. name - Genre object
GET /movies/directors/:directorName Get data about a director by name. directorName - Director object
POST /users Register a new user. - {"name": "John Doe"} New user object
GET /users/:userId Get data about a single user by user ID. userId - User object
PUT /users/:userId Update user info (name) by user ID. userId {"name": "New Name"} Updated user object
POST /users/:userId/favorites/:movieId Add a movie to the user's list of favorites by user ID and movie ID.
